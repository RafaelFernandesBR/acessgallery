# AcessGallery

AcessGallery é uma galeria de fotos pensada para pessoas cegas e com baixa visão: simples, direta e acessível. O app organiza suas fotos em álbuns, mostra detalhes das imagens e oferece ferramentas para backup e descrições por IA quando configuradas.

## Como o app ajuda

- Navegação simples entre álbuns e fotos, com foco na compatibilidade com leitores de tela.
- Descrições automáticas de imagens geradas por IA, para dar contexto sobre o que aparece nas fotos.
- Opções de backup.

O objetivo é tornar a experiência de ver e organizar fotos mais independente e acessível para quem é deficiente visual.

## Como usar (rápido)

1. Abra o app e permita acesso às suas fotos quando solicitado.
2. Use a lista de álbuns para navegar — o leitor de tela anunciará os itens.
3. Ao abrir uma foto, você encontrará opções para ver detalhes, compartilhar e gerar descrição com IA (se ativada).
4. Em `Configurações` você pode ativar/desativar descrições por IA, ajustar preferências de backup e configurar integrações.

## Instalação (para usuários)

instale o APK fornecido nas releases do repositório — após baixar, permita a instalação de apps desconhecidos no dispositivo e instale.

## Configurações de IA e privacidade

Se você ativar as descrições por IA, o app pode enviar imagens a serviços de IA (como APIs externas) para gerar texto descritivo. Essas integrações podem exigir chave de API — verifique as configurações do app para instruções de configuração.
